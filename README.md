# Portfolio (MANSHA.SYS)

A cyberpunk-themed interactive portfolio built with React and Framer Motion.

## 🚀 Features

- **Boot Sequence:** Retro-style terminal initialization.
- **Theming:** Dynamic dark/light mode with pixel backgrounds.
- **Sections:** Home, Projects, Experience, and Contact.
- **Animations:** Smooth page transitions and micro-interactions.

## 🛠️ Tech Stack

- **Framework:** React 19 + Vite
- **Styling:** Custom Vanilla CSS
- **Animations:** Framer Motion
- **Navigation:** React Router DOM

## ⌨️ Development

```bash
# Install
npm install

# Run Dev
npm run dev

# Build
npm run build
```
